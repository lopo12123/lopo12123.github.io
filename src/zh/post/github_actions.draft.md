# GitHub Actions

> 本页介绍`GitHub Actions`。

## 简介

`GitHub Actions`是**GitHub**平台提供的一项功能，用于自动化构建、测试和部署软件项目。它可以让我们在特定的事件（如`push` 、`pull request`事件等）或条件发生（如特定时间执行等）时执行自定义的工作流程。

**GitHub** 提供 **Linux**、**Windows**和**macOS**虚拟机来运行工作流程，也可以选择在自己的数据中心或云基础架构中托管自己的自托管运行器。
